# Отчёт сравнения: v2.28.0 → v2.28.1

Сравнение соседних индексированных версий (раздел 7 CLAUDE.md). Основа — `git diff v2.28.0..v2.28.1`.

- **От:** `v2.28.0` (`63cdf87`, 2025-05-20)
- **К:** `v2.28.1` (`a20891a`, 2025-08-26)

## Что изменилось (обзор)

Патч-релиз линии v2.28. Новых breaking changes нет. Подъём патч-версий Kubernetes и компонентов, перевод ArgoCD на установку по checksum, несколько новых Cilium/kubeadm-переменных из бэкпортов. Ansible-теги запуска не изменились.
