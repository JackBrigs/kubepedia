# containerd

## Summary

containerd is Kubespray's default container runtime (see
[[VARIABLE-CONTAINER_MANAGER]]). It is installed from a release archive whose
version is derived from the per-release checksums table, so it moves with the
Kubespray release. The generated runtime config is `/etc/containerd/config.toml`.

## Context

- Covers Kubespray `v2.29.0`–`v2.31.0`.
- Selected by `container_manager: containerd` (the default).
- Installed by the `roles/container-engine/containerd` role; runc and nerdctl are
  installed alongside it (their versions are derived the same way).

## Implementation

The version is derived, not pinned
(`roles/kubespray_defaults/defaults/main/main.yml`):

```yaml
containerd_version: "{{ (containerd_archive_checksums['amd64'] | dict2items)[0].key }}"
```

The value is the **first** (newest) key of `containerd_archive_checksums['amd64']`
in `roles/kubespray_defaults/vars/main/checksums.yml`. Concrete resolution per
tag:

| Kubespray | containerd version | main.yml line |
|-----------|--------------------|---------------|
| v2.29.0   | 2.1.4              | 81            |
| v2.30.0   | 2.2.1              | 83            |
| v2.31.0   | 2.2.3              | 83            |

containerd is installed from the release archive (not an image); the archive
checksum is selected by `image_arch` from `containerd_archive_checksums`.

## Configuration

- Version selection: `containerd_version`, `containerd_archive_checksums`
  (`kubespray_defaults`).
- Runtime config: `/etc/containerd/config.toml`, generated by the
  `roles/container-engine/containerd` role.
- Companions installed with containerd: `runc` (`runc_version`) and `nerdctl`
  (`nerdctl_version`), both derived as the first key of their respective checksum
  tables.
- Selection: [[VARIABLE-CONTAINER_MANAGER]] must be `containerd` (the default).

## Compatibility

- Kubespray `v2.29.0` → containerd `2.1.4`; `v2.30.0` → `2.2.1`; `v2.31.0` →
  `2.2.3`.
- Architecture: `containerd_archive_checksums` provides `amd64` and `arm64`; the
  archive is chosen by `image_arch`.
