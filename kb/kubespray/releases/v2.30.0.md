# Kubespray v2.30.0

## Summary

Kubespray `v2.30.0` (commit `f4ccdb5`) advances the Kubernetes support window by
one minor: adds `1.34`, drops `1.31`. Default Kubernetes `1.34.3`; supported
`1.32`–`1.34`.

## Implementation

Shipped versions (defaults), changes vs `v2.29.1` in bold:

| Component | Version |
|-----------|---------|
| Kubernetes (default / min) | **1.34.3** / **1.32.0** |
| etcd | **3.5.26** |
| containerd | **2.2.1** |
| runc | 1.3.4 |
| cni-plugins | 1.8.0 |
| Cilium | **1.18.6** |
| CoreDNS (default) | **1.12.1** |
| nerdctl | 2.2.1 |
| MetalLB | 0.13.9 |
| kube-vip (deployed) | **v1.0.3** |

## Upgrade Notes

- **Minor-window shift:** Kubernetes `1.31` is no longer supported; upgrade one
  minor at a time.
- kube-vip: from `v2.30.0` the image tag is derived from `kube_vip_version`
  (`1.0.3`), fixing the earlier literal-tag mismatch.
- kubeadm config unchanged (`v1beta4`).

## Compatibility

- Kubernetes `1.32`, `1.33`, `1.34`.
