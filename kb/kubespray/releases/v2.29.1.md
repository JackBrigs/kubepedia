# Kubespray v2.29.1

## Summary

Kubespray `v2.29.1` (commit `0c6a295`) is a patch release over `v2.29.0`: same
Kubernetes minor window (`1.31`–`1.33`) with refreshed patch versions. Default
Kubernetes `1.33.7`.

## Upgrade Notes

- Patch release: no minor-window change; upgrading from `v2.29.0` refreshes patch
  versions and the default Kubernetes version (1.33.5 → 1.33.7).
- kubeadm config unchanged (`v1beta4`).

## Compatibility

- Kubernetes `1.31`, `1.32`, `1.33` (patch ranges bumped: 1.31.0–1.31.14,
  1.32.0–1.32.10, 1.33.0–1.33.7).
