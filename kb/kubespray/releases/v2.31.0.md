# Kubespray v2.31.0

## Summary

Kubespray `v2.31.0` (commit `1c9add4`) is the newest indexed release: adds
Kubernetes `1.35`, drops `1.32`. Default Kubernetes `1.35.4`; supported
`1.33`–`1.35`. Notably, etcd `3.6` is introduced for Kubernetes `1.35`.

## Upgrade Notes

- **etcd 3.6** enters via a per-minor selection ceiling raised to `< 3.7` for
  Kubernetes `1.35` only (older minors stay on `3.5.x`).
- **runc** major bump `1.3 → 1.4`; **Cilium** minor bump `1.18 → 1.19`.
- **kubeadm config:** the `kubeadm_config_api_version` variable and the `v1beta3`
  fallback template were **removed**; the template is pinned to `v1beta4`
  ([[CONFIG-KUBEADM_CONFIG_API_VERSION]]).
- The legacy `master` run-tag is fully gone (renamed to `control-plane`,
  completed here in the upgrade playbook).
- Minor-window shift: Kubernetes `1.32` no longer supported.

## Compatibility

- Kubernetes `1.33`, `1.34`, `1.35`.
