# Kubespray v2.28.1

## Summary

Kubespray **v2.28.1** ships Kubernetes **1.32.8** with the component set below. It is part of
the extended `v2.27.0`–`v2.31.0` coverage (baseline research is v2.29.0). Patch release: bug fixes only, same K8s 1.

## Implementation

Component versions (tag `v2.28.1`):

| Component | Version |
|-----------|---------|
| Kubernetes default | 1.32.8 |
| etcd | 3.5.22 |
| containerd | 2.0.6 |
| cri-o | 1.32.0 |
| runc | 1.2.x |
| cni-plugins | 1.4.1 |
| calico | 3.29.5 |
| cilium | 1.17.7 |
| CoreDNS | 1.11.3 |
| ingress-nginx | 1.12.1 |
| argocd | 2.14.5 |
| helm | 3.16.4 |

## Upgrade Notes

- **Patch** release: bug fixes only, same K8s 1.32 line as v2.28.0.
- Fixes: Cilium install/templating & loadBalancer.mode, Cilium BGP control-plane config, calico BGP config, external etcd member removal, PodSecurity `use_default:false`.

## Compatibility

- Kubernetes line: **1.32.8** (minimum `1.30.x`).
- Component versions move per the selection mechanism ([[CONCEPT-COMPONENT_VERSION_SELECTION]]);
  concrete values above are for tag `v2.28.1`.
- Upgrade **one Kubespray minor at a time** ([[UPGRADE-KUBESPRAY_SEQUENTIAL]]).
