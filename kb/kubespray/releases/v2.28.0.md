# Kubespray v2.28.0

## Summary

Kubespray **v2.28.0** ships Kubernetes **1.32.5** with the component set below. It is part of
the extended `v2.27.0`–`v2.31.0` coverage (baseline research is v2.29.0). ACTION REQUIRED — remove the leading `v` from all explicit component versions (notably `kube_version`).

## Implementation

Component versions (tag `v2.28.0`):

## Upgrade Notes

- **ACTION REQUIRED — remove the leading `v`** from all explicit component versions (notably `kube_version`).
- **Removed:** Krew install support; `etcd_kubeadm_enabled` (deprecated) removed.
- **Renames:** role `kubespray-defaults` → `kubespray_defaults`; `bootstrap-os` → `bootstrap_os`.
- Cilium install now uses the **Cilium CLI** (not a Jinja template); `cilium_identity_allocation_mode` default → `crd`; kube-proxy nftables mode.
- K8s 1.32 hashes + containerd 2.0.x; control-plane memory requirement raised to 2GB. **Last RHEL 8 version.**

## Compatibility

- Kubernetes line: **1.32.5** (minimum `1.30.x`).
- Component versions move per the selection mechanism ([[CONCEPT-COMPONENT_VERSION_SELECTION]]);
  concrete values above are for tag `v2.28.0`.
- Upgrade **one Kubespray minor at a time** ([[UPGRADE-KUBESPRAY_SEQUENTIAL]]).
