# Kubespray v2.27.1

## Summary

Kubespray **v2.27.1** ships Kubernetes **1.31.9** with the component set below. It is part of
the extended `v2.27.0`–`v2.31.0` coverage (baseline research is v2.29.0). Patch release: bug fixes only, same K8s 1.

## Upgrade Notes

- **Patch** release: bug fixes only, same K8s 1.31 line as v2.27.0.
- Fixes: control-plane reconfiguration on upgrades, kubeadm v1beta4 UpgradeConfiguration, calico RBAC (kubecontrollersconfigurations list).

## Compatibility

- Kubernetes line: **1.31.9** (minimum `1.29.0`).
- Component versions move per the selection mechanism ([[CONCEPT-COMPONENT_VERSION_SELECTION]]);
  concrete values above are for tag `v2.27.1`.
- Upgrade **one Kubespray minor at a time** ([[UPGRADE-KUBESPRAY_SEQUENTIAL]]).
